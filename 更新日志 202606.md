# 更新日志 202606

## 2026-06-07

### 模型参数与思考模式

- 调整主模型调用参数：思考模式开启时不再自动过滤 `temperature`。
- 删除主模型路径中的 `EXECUTOR_OMIT_TEMPERATURE` 死配置，避免误以为思考模式仍有隐藏过滤开关。
- 更新 OpenAI 兼容适配层说明，明确思考模式会继续传递 `temperature`。

### 归档目录

- 归档目录默认展开，但不会因展开/折叠自动加载归档会话。
- 归档区只显示“加载归档目录”按钮，只有点击按钮才请求归档会话列表。
- `/sessions` 增加 `X-Archived-Count` 响应头，在不加载归档列表的情况下也能显示真实归档数量。
- 前端在加载归档前使用响应头数量，加载后改用实际归档列表长度。

### 上下文压缩

- 修复首次压缩摘要格式异常时缺少明确重试/降级提示的问题。
- 修复第二次压缩可能直接显示“摘要轮次已用尽”的问题，保证摘要轮次下限并区分触发原因。
- 优化压缩失败、无可摘要前缀、轮次耗尽等状态文案，让界面能更准确说明当前发生了什么。

### Subagent 结果续接

- 修复自动续接无法识别 subagent 结果是否已经被读取的问题。
- `task(collect_result=true)` 显式读取 subagent 结果后，会清理对应 pending 结果，避免后续续接重复提示或重复处理。
- 新增按 agent id 清理 pending subagent results 的会话管理能力。

### 前端源码与构建产物

- 明确前端源码目录为 `frontend/`，构建产物为 `app/templates/dist/`。
